Módulo para Magento Normatiza Brasil
=================
O módulo Normatiza Brasil é uma iniciativa para traduzir e adaptar os módulos nativos do cadastro de usuários do Magento.

Oque esse módulo faz?
=================
Com o módulo Normatiza Brasil será introduzido no cadastro de usuário do Magento novos campos como:

* Campo tipo pessoa (Física ou Jurídica)
* Tradução dos Campos do formulário
* Formatação dos campos por máscara. Exe: Telefone com 9 ou 8 dígitos (99) ?9999-9999
* Busca do endereço por ajax direto do site dos correios
* Campo IE(Inscrição Estadual) para geração da NFe


Atenção!!!
=================
O módulo ainda encontra-se em fase de desenvolvimento! Ou seja, ainda não funiona e foi colocado no Git para colaborações de desenvolvedores!
